# 🎨 AI Data Assistant – UI Preview

> 👀 See how the application looks and works before running it.

---

## 💬 Chat Interface

The application provides a clean, modern chat interface where users interact with the AI to perform machine learning tasks.

* Guided workflow through conversation
* Minimal and structured UI
* Smooth chat experience

---

## 📊 Analytics Dashboard

After training the model, users can view detailed analytics and insights.

* Feature importance visualization
* Model performance metrics
* Clean dashboard layout

---

## 🧠 Session State Panel

The right panel keeps track of everything in real time:

* Selected model
* Target column
* Task type
* Metrics (Accuracy, F1, etc.)

---

## ⚡ Key UI Highlights

* Modern dark theme
* Clean layout with sidebar navigation
* Chat + Analytics workflow separation
* Smooth and responsive interaction
* Structured information panels

---

## 🚀 User Flow

1. Upload dataset
2. Chat guides ML process
3. Model gets trained
4. Enter inputs
5. Get prediction
6. View analytics dashboard
